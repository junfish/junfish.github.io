总统是靠不住的
林达 / 生活·读书·新知三联书店 / 2004-08 / 21.80
2022-07-04 读过 / / ★★★★☆ 推荐
美国的制度设计从来不仰仗某种权威来伸张正义，相反，大多数人认为权威会过度膨胀进而侵蚀公平正义，历史经验也告诉我们确实是这样。因此，美国人不惜以损失效率和权力制衡来换取社会与国家的长治久安，并且国民对这种代价也甘之如饴。我觉得这是一种历史的进步，毕竟，人民太需要依法治国了。
历史深处的忧虑
林达 / 生活·读书·新知三联书店 / 1997-5 / 19.00元
2022-04-22 读过 / / ★★★★☆ 推荐
看罢初篇，结合当下中美竞争的大时代背景和我在美国求学生活的经历，其实对很多国人同胞，我想说不要总觉得好像美国发展到现在开始出现了诸如此类的问题，其实美国一直是那样，只是中国的发展使得国人有了自信去用别样的视角和眼光去看待一些问题。要知道，美国也是一直在发展的，发展速度不好去衡量，但你要是真觉得这样的一个国家机器正走向衰落，那可能还需要沉下心来多看些书多走些路。我们还没吃几年的白米饭，好好潜下心来学人所长补己之短才是王道，师夷长技以制夷，与君共勉。
给青年的十二封信
朱光潜 / 广西师范大学出版社 / 2004-12 / 12.80
2021-07-15 读过 / / ★★★★★ 力荐
看时悔不能早些年遇到这本书信集，但转念一想，早些年的我也无法把朱先生这些鞭辟入里的谆谆教导内化于心，便也释然了。朱先生由心运思，心之所至，思之所开。体验生活的本质与拥有深刻的思想原来可以如此有机地统一。大家手笔，可见一斑，让人不禁想拜读先生其他的学术著作。也许，以实际的生活体验为基，遵从内心的情感导向，世间很多的纷争与矛盾会显得开阔与有趣，人生的很多选择与踯躅也显得简单起来。这对于我等极端推崇理性理想国的愚人而言，真真切切是善意的提醒。
培根随笔集
弗兰西斯·培根 / 王义国 / 中央编译出版社 / 2010-4 / 12.00元
2021-06-03 读过 / / ★★★★★ 力荐
四百年前的著作读起来依然能让人感受到字里行间智慧的流动。始读时，你会流连于培根对修辞的运用与掌控，复杂深奥的概念在比喻中被通俗化。你还会惊讶于他如百科全书般的旁征博引，感慨原来学识与人情事故竟可以如此的相通。最后你将享受到那种醍醐灌顶，思维进阶的快感。

今日简史
[以] 尤瓦尔·赫拉利 / 林俊宏 / 中信出版社 / 2018-8 / 68
2020-06-26 读过 / / ★★★☆☆ 还行
《今日简史》难免有些乘热度炒冷饭之嫌，好在作者在写作意图上花了一些笔墨修饰。因此，本书单纯定位为自由辩论、公众对话与关心当下，读起来比较酣畅，也免去了一些口诛笔伐。只是在成书结构的整体安排上，早已缺失了前两部作品所具有那种整体感与连续感。议题上比写在前面的两本更广，无所不谈，观点阐述也更加多元，却因为缺失了那根贯穿全书的线而显得像是一本杂文集。
未来简史
[以色列] 尤瓦尔·赫拉利 / 林俊宏 / 中信出版社 / 2017-2 / 68.00元
2020-05-20 读过 / / ★★★☆☆ 还行
《未来简史》算是秉承《人类简史》逻辑性不强的特点，并增加炒冷饭这个槽点。本书从第二章到第七章花大笔墨以人文主义的形成做为新的线索，把人类简史做了一次重解读。在我看来，作者完美地证明了一个论点——历史就是个任人打扮的小菇凉。当然，作者跨学科的深入浅出也是挺让人佩服的。尤其涉及到对机器学习与人工智能的理解与论述，因为恰好涉及到本命专业知识领域，我可以很负责任的说，作者观点符合目前业界弱人工智能蓬勃发展的趋势。如果作者认为某个机器智能可以超越人类个体，那本书的其他论述在我心中就也要大打折扣了。总体评价，作者在书中抛出了一个有趣的未来学观点：智人追求永生和幸福，并最终会成为智神。然后用一些无法证实与证伪的论据使得本书自圆其说，读罢有开拓思路的快感，却也因为看过《人类简史》之后了解其套路而有些乏了。
人类简史
[以] 尤瓦尔·赫拉利 / 林俊宏 / 中信出版社 / 2017-2-1 / 68.00元
2020-05-08 读过 / / ★★★☆☆ 还行
作者以智人历史的认知革命、农业革命和科学革命为时间线娓娓道来自己的历史观、人类观，脑洞大开地构建了宏大的人类发展历程及其个中缘由。但很多观点只流于陈述，逻辑勉强自洽，却没有充分的依据与论证。譬如作者认为农业革命非智人个体所愿，且个体为其所累，看罢本书也未能有这个观点的严格论证，有的只是逻辑自洽的旁征博引。看起来似乎是智人为了社会发展放弃了个体幸福，然而，作者却又根据此论述推导出社会整体的进步会导致个体幸福感的降低，互为因果的论证比比皆是。在我看来，本书的意义在于作者从一个新颖角度（人类历史基于虚构建立）去解读人类历史，并从宏观上提出很多独到的见解。能让人看得酣畅淋漓，却不能缜密思考。
邓小平时代
【美】傅高义 (Ezra.F.Vogel) / 冯克利 / 生活·读书·新知三联书店 / 2013-1-18 / 88.00元
2020-05-06 读过 / / ★★★★★ 力荐
中国当代社会的得与失都在本书中有迹可循。邓小平将个人完美的政治手段和智慧运用于实践，完成中国历史上最为深刻的政治变革、社会变革与思想变革。他为中国未来阶段的稳定发展营造了一个无与伦比的内部环境，这种内部环境与外部国际环境相协调、相适应。立足于如今的时代节点来看，尽管一些考量和设计的影响得失并存，但邓小平的战略布局无疑是那个时代的最优解。看罢此书，深深为邓公务实的作风、开放的思想和前瞻的眼光所折服。
