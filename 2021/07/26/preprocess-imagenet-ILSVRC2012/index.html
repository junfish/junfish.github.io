2021-07-26 计算机视觉 ImageNet, Preprocess, 预处理 0 0 Comments Word Count: 671(words) Read Count: 2(minutes)

本文目的

鉴于目前网络上并无相关从 imagenet 官网下载竞赛数据到训练的相关教程，本文提供从获取数据到训练的全步骤讲解。

下载并解压数据

  1. 进入官网下载页面 https://image-net.org/challenges/LSVRC/2012/2012-downloads.php 获取下载链接。

  2. 利用 wget命令下载数据（一共四个数据文件）。

  3. 利用tar命令解压缩数据包放入指定文件夹

    首先尝试解压缩部分文件探索数据文件目录结构：tar -xvf ./ILSVRC2012_img_train_t3.tar -C test。然后，通过bash命令批处理删除生成的文件：find ./test -name '*.JPEG' -type f -print -exec rm -rf {} \;。
